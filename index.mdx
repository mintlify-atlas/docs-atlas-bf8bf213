## What you'll find here

<Accordion title="Laravel tips and best practices">
  Discover shortcuts and optimization techniques for:
  - **Eloquent ORM** - Query optimization, dynamic methods, scopes, and modern PHP attributes
  - **Collections** - Higher-order messages, filtering, and generation methods
  - **Blade** - Template shortcuts, directives, and component techniques
  - **Validation** - Relative date validation and image dimension rules
  - **Helpers** - String manipulation, number formatting, and date handling
  - **Artisan Commands** - Model generation shortcuts
  - **Tutorials** - Complete implementations like slug automation and custom email verification
</Accordion>

<Accordion title="PHP language features">
  Modern PHP techniques including:
  - Array destructuring in loops
  - Spread operator for merging arrays
  - `isset()` with multiple parameters
  - `array_filter()` and `array_map()` shortcuts
</Accordion>

<Accordion title="CSS techniques">
  Learn about modern CSS features:
  - Native CSS nesting without preprocessors
  - Using `&` for parent selectors
  - Nested pseudo-classes and compound selectors
</Accordion>
