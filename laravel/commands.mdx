## Générer une migration avec l'option `-mcfsr`

Générer un model avec sa migration, son Controller, sa Factory et son Seeder en une seule commande avec les options `-mcfsr` 🚀🚀🚀

```bash
php artisan make:model Post -mcfsr

# \Models\Post.php] created successfully.
# \factories\PostFactory.php] created successfull
# \migrations\2025_09_18_170933_create_posts_table.php] created successfully.
# \seeders\PostSeeder.php] created successfully.
# \Controllers\PostController.php] created successfully.
```
